# Gemma Coding Test Seed Plan

## Goal

Create a public GitHub repo `jakehulberg/gemma-coding-test` containing enough detail for Gemma to build Agent Task Arena task-by-task in fresh local Hermes sessions.

## Approach

Seed the repo with docs only:

- Product requirements document
- Sequential implementation task list
- Gemma operating workflow
- Evaluation rubric
- Copy/paste task packets

The implementation itself should be performed later by Gemma, one task per fresh session, with one commit/push per task.

## Validation

- Repo exists and is public.
- Docs are committed and pushed.
- Jake receives exact commands for running Gemma.
